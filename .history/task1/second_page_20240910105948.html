HTML CSS JavaScript

HTML

  • Нужен, чтобы размещать на веб-странице элементы: текст, картинки, таблицы и видео
  • выстраивает визуальный фундамент сайта: указывает, где располагаются элементы, какой у них будет базовый дизайн, откуда брать стили для элементов и скрипты
  • Состоит из тегов — команд, которые указывают браузеру, как отображать помещённый в них контент
  • Файлы обычно имеют расширение «.html» или «.htm»
  • В HTML-файле можно задавать:
    • гиперссылки
    • списки
    • формы
    • разметку страницы
    • таблицы
    • абзацы
    • картинки
    • видео
    • заголовки

CSS

  • Определяет фон, прозрачность, цветовую гамму и размеры, изменение при взаимодействии - это простой способ задать стили для разных элементов сайта
  • Упрощает процедуру разработки и дальнейшего редактирования страниц
  • Создает богатое разнообразие вариантов визуализации контента
  • Создает возможность настраивать разные стили в рамках одной страницы
  • Типичная запись состоит из трёх частей: селектора, свойства и значения

JavaScript

  • Формирует поведение объектов страницы
  • В браузере для JavaScript доступно всё, что связано с манипулированием веб-страницами, взаимодействием с пользователем и веб-сервером
  • Полная интеграция с HTML/CSS
  • Поддерживается всеми основными браузерами и включён по умолчанию
  • Поддерживает ООП
